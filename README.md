# pulse-oximeter-rn
A small React Native application for the Nonin 3230 pulse oximeter

## Setup and start
Follow the Getting Started guide on the React Native website:
https://facebook.github.io/react-native/docs/getting-started.html

```bash
$ yarn && yarn start
```
